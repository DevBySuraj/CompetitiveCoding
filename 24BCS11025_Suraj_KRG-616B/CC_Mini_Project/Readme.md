# Smart Task Manager (DSA Project)

## 🚀 Features
- Priority Queue using Min Heap
- Add / Delete / Execute tasks
- Search and Filter
- Persistent storage (LocalStorage)

## 🧠 DSA Used
- Min Heap (Priority Queue)
- Time Complexity:
  - Insert: O(log n)
  - Delete: O(log n)

## 💻 Tech Stack
- HTML, CSS, JavaScript

## 🎯 Use Case
Simulates real-world task scheduling systems like OS process scheduling.
